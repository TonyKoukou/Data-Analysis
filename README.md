# Data Analysis


## Περιγραφή
Python πρόγραμμα που διαβάζει δεδομένα από CSV, υπολογίζει μέσους όρους και δημιουργεί γράφημα bar chart.


## Οδηγίες Χρήσης
1. Βεβαιώσου ότι έχεις εγκατεστημένη Python 3.
2. Άνοιξε το πρόγραμμα:cmd
3. Γράψε την εντολή:pip install pandas matplotlib
4. Δημιούργησε αρχείο data.csv με μορφή:
	Name,Score1,Score2,Score3
	Maria,18,19,20
	John,15,17,16
	Anna,20,20,19
5. Γράψε cd και την τοποθεσία του αρχείου π.χ cd C:\Users\Tony\Data Analysis
6. Γράψε την εντολή: data_analysis.py
